# NG FOOD

Este projeto foi criado como um desafio do curso de Front-end do IGTI.
<br><br>

## Executar projeto localmente

- Instalar o node
- Instalar o Angular CLI
- Baixar o pojeto
- descompactá-lo e entrar na pasta com o cmd
- instalar as dependências

  ```
  > npm install
  ```

- subir o server

  ```
  > ng serve
  ```
